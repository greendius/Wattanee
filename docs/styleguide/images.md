## Image sizes

Just a reference guide. Images' width should always be - `100% width`.

***

Square 75 x 75<br />
Thumbnail 100 x 80<br />
Small 240 x 160<br />
Medium 500 x 335<br />
Large 850 x 400<br />
Banner 1130 x 450<br />

***

### Square

{% example html %}
<div class="img-square">
  <img src="http://loremflickr.com/75/75/flower" />
</div>
{% endexample %}
